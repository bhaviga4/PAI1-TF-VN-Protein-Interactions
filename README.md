# PAI-1, Tissue Factor, and Vitronectin Protein Interactions

## Project Overview

This project investigates the molecular interactions among Plasminogen Activator Inhibitor-1 (PAI-1), Tissue Factor (TF), and Vitronectin (VN) using computational biology and bioinformatics approaches. The study aims to understand the structural and dynamic behavior of these proteins and their potential roles in disease-related pathways.

## Objectives

* Analyze protein-protein interactions between PAI-1, TF, and VN.
* Perform molecular docking studies to identify binding interfaces.
* Evaluate complex stability using molecular dynamics simulations.
* Investigate structural changes and interaction patterns through trajectory analysis.

## Methodology

### 1. Protein Structure Preparation

* Retrieval of protein structures from public databases.
* Structure cleaning and preprocessing.

### 2. Molecular Docking

* Protein-protein docking performed to predict interaction complexes.
* Selection of best-scoring docking poses for further analysis.

### 3. Molecular Dynamics Simulation

* System preparation using CHARMM-GUI.
* Simulation performed using GROMACS.
* Trajectory generation and analysis.

### 4. Trajectory Analysis

* RMSD (Root Mean Square Deviation)
* RMSF (Root Mean Square Fluctuation)
* Radius of Gyration (Rg)
* Hydrogen Bond Analysis
* Protein Interaction Stability Assessment
